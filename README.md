# Wine-Classification-Using-Decision-Tree-and-Logistic-Regression
Machine Learning project that compares Decision Tree and Logistic Regression algorithms on the Wine Dataset using Python and Scikit-Learn.
# Wine Classification Using Machine Learning

## Overview

This project compares the performance of two Machine Learning algorithms:

- Decision Tree Classifier
- Logistic Regression

The Wine Dataset is used to classify wine samples based on their chemical properties.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset Features

- Alcohol
- Malic Acid
- Ash
- Magnesium
- Flavanoids
- Color Intensity
- Hue
- Proline

and several other wine attributes.

## Models Used

### Decision Tree
A tree-based classification algorithm that splits data into decision nodes.

### Logistic Regression
A statistical classification algorithm used for multi-class prediction.

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
